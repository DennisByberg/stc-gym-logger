# 🧾 Useful Commands

A small collection of commands used while building this project.

## 📱 Ionic / Angular

### Start the mobile app locally

Run this from the Ionic app directory:

```bash
cd apps/mobile
ionic serve
```

This starts the development server for the mobile frontend.

Default local URL:

```text
http://localhost:8100
```
